========================================
  這個網站怎麼改？（先看這張）
========================================

網站上會顯示的文字，全部都在下面幾個檔裡。
這些檔的副檔名雖然是 .md 或 .yml，但它們「就是純文字檔」，
用「記事本」或 VS Code 打開就能改，跟 txt 一模一樣。

※ 為什麼不能改成 .txt？
  因為網站程式只認得 .md（內容）和 .yml（設定），
  改成 .txt 網站會壞掉、不再更新。副檔名要留著，但編輯方式跟 txt 相同。


怎麼打開這些檔？（三選一）
------------------------------------------
(A) 在檔案上按右鍵 →「開啟方式」→ 選「記事本」。
(B) 用 VS Code：把整個資料夾拖進 VS Code，左邊點檔案就能改。
(C) 最簡單、免軟體：直接到 GitHub 網站點那個檔 → 按右上角鉛筆圖示改 → 按 Commit。


想改什麼 → 開哪個檔
------------------------------------------
1. 中文首頁的全部內容（著作、經歷、獎項、學歷…）
   →  _pages/about.md

2. 英文版頁面的全部內容
   →  _pages/about-en.md

3. 左邊側欄（姓名、簡介、地點、單位、Email、GitHub、Instagram）
   →  _data/profile.yml

4. 上方那兩個「中文 / English」按鈕的文字
   →  _data/navigation.yml

5. 換大頭照
   →  把照片命名成 profile.png，放進 images/ 資料夾覆蓋

6. 網站配色（背景、標題、文字顏色）
   →  _sass/theme/_my_scheme_light.scss（改最上面幾個色碼）


改完之後
------------------------------------------
存檔 → 用你的 git 做 commit + push → 等 GitHub 出現綠勾 → 網站就更新了。
（若用上面 (C) 直接在 GitHub 改，按 Commit 後就會自動更新，不用另外 push。）
